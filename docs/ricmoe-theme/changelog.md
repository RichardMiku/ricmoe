## 迁移指南

### 从 v0.9.x 升级到 v1.0.0

1. **更新依赖**:
```bash
npm update
```

2. **迁移 Links 数据**:
```typescript
// 旧格式
const links = [
  { title: 'Site', url: 'https://example.com' }
];

// 新格式  
const links: LinkProps[] = [
  {
    title: 'Site',
    url: 'https://example.com', 
    category: 'friends',
    tags: ['blog'],
    status: 'active'
  }
];
```
