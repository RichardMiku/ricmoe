# RICMOE

## 欢迎！

欢迎访问 RICMOE！

RICMOE 实现了从 hexo 向 docusaurus 的迁移，旨在成为一个更现代化的文档和博客平台。

:::info Care
这意味着，这是一个全新的开始，RICMOE 将不再迁移原 hexo 的内容。
:::

## 介绍一下！

### 为什么从 hexo 迁移到 docusaurus？

1. **现代**：Docusaurus 提供了更现代化的界面和功能，使文档和博客的创建和管理变得更加高效和便捷。
2. **支持**：Docusaurus 背靠 MetaOpenSource，拥有强大的社区支持和持续的更新迭代。
3. **丰富**：Docusaurus 使用 React 构建，支持诸多组件和插件，使功能更加丰富。
4. **高自定义**：Docusaurus 提供了高度的自定义能力，用户可以根据自己的需求自由调整和扩展功能。

### MDX 新特性

> **Docusaurus提供了 `MDX` 支持，这使得在使用 Markdown 进行创作时能引入更丰富的组件和交互。**

多样的信息块：

:::info 这是一个信息提示
:::
:::tip 这是一个想法提示
:::
:::danger 这是一个警告提示
:::
:::note 这是一个注释提示
:::
:::caution 这是一个注意提示
:::

支持引入 React 组件：

```jsx title="一个简单的 Button 示例"
<button className='button button--outline button--primary margin-bottom--lg' onClick={() => alert('Hello, welcome to RICMOE!')}>Click Me</button>
```
